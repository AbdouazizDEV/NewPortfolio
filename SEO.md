# Guide SEO - Portfolio Abdou Aziz Diop

## Optimisations SEO Implémentées

### 1. Métadonnées HTML
- ✅ Titre optimisé avec mots-clés pertinents
- ✅ Description meta détaillée (150-160 caractères)
- ✅ Mots-clés ciblés pour le référencement
- ✅ Balises meta géographiques (Dakar, Sénégal)
- ✅ Balises robots pour l'indexation
- ✅ Canonical URL pour éviter le contenu dupliqué

### 2. Open Graph (Facebook, LinkedIn)
- ✅ og:title, og:description, og:image
- ✅ og:url, og:type, og:locale
- ✅ Dimensions d'image optimisées (1200x630)

### 3. Twitter Cards
- ✅ twitter:card (summary_large_image)
- ✅ twitter:title, twitter:description
- ✅ twitter:image avec alt text
- ✅ twitter:creator et twitter:site

### 4. Données Structurées (Schema.org)
- ✅ Person Schema (informations personnelles)
- ✅ WebSite Schema (site web)
- ✅ ProfessionalService Schema (services)

### 5. Fichiers de Configuration
- ✅ robots.txt (gestion des crawlers)
- ✅ sitemap.xml (plan du site)
- ✅ site.webmanifest (PWA)

### 6. Performance
- ✅ Preconnect pour les ressources externes
- ✅ DNS-prefetch pour améliorer les temps de chargement
- ✅ Optimisation des fonts

## Prochaines Étapes Recommandées

### 1. Analytics et Suivi
- Installer Google Analytics 4
- Configurer Google Search Console
- Ajouter des événements de tracking

### 2. Performance
- Optimiser les images (WebP, lazy loading)
- Minifier le CSS et JavaScript en production
- Implémenter le lazy loading des composants

### 3. Contenu
- Ajouter un blog avec des articles techniques
- Créer des pages dédiées pour chaque projet
- Ajouter des témoignages clients

### 4. Liens Externes
- Obtenir des backlinks de qualité
- Partager sur les réseaux sociaux
- Créer un profil sur des plateformes de développeurs

### 5. Local SEO
- Créer un profil Google Business (si applicable)
- S'inscrire sur des annuaires de développeurs
- Optimiser pour les recherches locales "développeur Dakar"

## Commandes Utiles

### Vérifier le SEO
```bash
# Tester avec Lighthouse
npm run build
npm run preview
# Ouvrir Chrome DevTools > Lighthouse > SEO
```

### Soumettre le sitemap
1. Aller sur Google Search Console
2. Sitemaps > Ajouter un nouveau sitemap
3. Entrer: https://abdouazizdiop.dev/sitemap.xml

### Tester les données structurées
- Google Rich Results Test: https://search.google.com/test/rich-results
- Schema.org Validator: https://validator.schema.org/
